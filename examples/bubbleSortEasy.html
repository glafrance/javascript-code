Bubble sort is a simple sorting algorithm that repeatedly steps through a list of items to be sorted. It compares each pair of adjacent items and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted. This comparison sort algorithm is named for the way smaller elements 'bubble' to the top of the list.
